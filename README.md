# Airline-System
Python-based airline data analysis system that processes CSV datasets to generate insights and visualisations. 

## How to Run  
1. Download the repository
2.  Open the folder in VS Code or any Python IDE
3.  Ensure Python is installed
4.  Run the main file: w2144658
5.  Follow the prompts in the terminal 6. When asked  to enter a code and year choose either (LHR 2025) OR (CDG 2021)
